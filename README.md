# rsap-demo-website
Demo website for RSAP – Riders Safety Advocates of the Philippines, showcasing responsive design, TailwindCSS, AOS animations, and interactive UI components for portfolio purposes.

This is a **demo website** for RSAP – Riders Safety Advocates of the Philippines, showcasing modern, responsive front-end skills using **HTML5, TailwindCSS, and JavaScript**.

---

## Features
- Responsive layout for desktop and mobile
- Hero section with call-to-action buttons
- Join the Advocacy modal form
- Founder section with circular profile image
- National Programs section with cards and hover effects
- Scroll-based animations using AOS
- Interactive buttons with hover transforms and shadow effects

> **Note:** This is a portfolio project only, not affiliated with RSAP.
